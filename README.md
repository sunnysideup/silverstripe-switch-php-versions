# what it does:

switch PHP versions with a simple command from anywhere:

```shell
php-switch 7.3
```

OR 

```shell
php-switch 7.1
```


# how to install:

1. git clone this project
2. type: 
```shell
sudo bash silverstripe-switch-php-versions/install.sh
```
3. `rm silverstripe-switch-php-versions -rf`
Bob still is your uncle.
